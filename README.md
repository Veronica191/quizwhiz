1.  QuizWhiz (React Quiz App)

QuizWhiz is an interactive quiz application built with React that makes learning engaging and fun. Users can answer multiple-choice questions, receive instant feedback, and view their scores in real time.

This project focuses on practicing React fundamentals, component-based architecture, and state management while building a real-world application.

2. Features

* Multiple-choice quiz questions

* Real-time score updates using React state

* Instant feedback on selected answers

* Restart quiz functionality

* Final results summary and answer review

* Clean and responsive UI

* App Demo (How It Works)

The user starts a quiz

Questions are rendered dynamically using React components

The user selects an answer

State updates instantly to reflect correctness and score

At the end, results are displayed

The user can restart the quiz and try again

⚠️ Note: Minor bugs may still exist. These are part of the development process and are being improved.
